# Bellabeat Smart Device Usage Analysis

## Overview
Analysis of Fitbit tracker data to identify wellness trends 
and inform Bellabeat's marketing strategy.

## Tools Used
- Python (Pandas) — data cleaning & processing
- MS Excel
- VS Code
- Power BI — dashboard & visualisation

## Dataset
Fitbit Fitness Tracker Data — Kaggle (CC0 Public Domain)

## Key Findings
- Only 41.67% of users are highly engaged
- Only 20.83% meet the 10,000 steps/day goal
- No strong correlation between activity and sleep duration
- 54% of users don't meet recommended sleep hours
